Shortcuts make you much faster. On Mac use **Cmd**; on Windows/Linux use **Ctrl**.

> Shortcuts are paused while you're **typing in a text field** and while the **preview** is open, so they never fire by accident.

## Tools

Press a letter to switch tools instantly.

| Key | Tool |
| --- | --- |
| **V** | Select |
| **T** | Text |
| **I** | Image |
| **R** | Rectangle |
| **L** | Line |
| **O** | Shape |
| **B** | Barcode |
| **Q** | QR Code |
| **G** | Table |
| **E** | Repeat |
| **C** | Condition |
| **F** | Frame |
| **S** | Signature |
| **Esc** | Back to Select |

## Editing

| Shortcut | Action |
| --- | --- |
| **Cmd/Ctrl + Z** | Undo |
| **Cmd/Ctrl + Shift + Z** | Redo |
| **Cmd/Ctrl + Y** | Redo (alternative) |
| **Cmd/Ctrl + C** | Copy selection |
| **Cmd/Ctrl + V** | Paste |
| **Cmd/Ctrl + D** | Duplicate selection |
| **Delete** or **Backspace** | Delete selection |

## Grouping

| Shortcut | Action |
| --- | --- |
| **Cmd/Ctrl + G** | Group selection |
| **Cmd/Ctrl + Shift + G** | Ungroup |

## Stacking order (one item selected)

| Shortcut | Action |
| --- | --- |
| **Cmd/Ctrl + ]** | Bring forward |
| **Cmd/Ctrl + [** | Send backward |
| **Cmd/Ctrl + Shift + ]** | Bring to front |
| **Cmd/Ctrl + Shift + [** | Send to back |

## Moving (nudging) a selection

| Shortcut | Action |
| --- | --- |
| **Arrow keys** | Move 1 pixel |
| **Shift + Arrow keys** | Move 10 pixels |

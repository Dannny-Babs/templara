`@templara/templates` provides ready-made business document templates and matching sample data. It is useful for demos, tests, local projects, golden renders, and onboarding flows.

## Main exports

| Export | Purpose |
| --- | --- |
| `invoiceTemplate`, `invoiceSampleData` | Invoice starter |
| `shipmentBolTemplate`, `shipmentBolSampleData` | Shipment bill of lading starter |
| `receiptTemplate`, `receiptSampleData` | Receipt starter |
| `payStubTemplate`, `payStubSampleData` | Pay stub starter |
| `shippingLabelTemplate`, `shippingLabelSampleData` | Shipping label starter |

## Use a starter

```ts
import { invoiceSampleData, invoiceTemplate } from '@templara/templates';

const project = {
  name: 'Invoice',
  template: structuredClone(invoiceTemplate),
  data: structuredClone(invoiceSampleData),
};
```

Clone before editing. Starter templates should be treated as immutable seeds.

## Render a starter

```tsx
import { renderDocument } from '@templara/renderer';
import { DocumentPreview } from '@templara/react-renderer';
import { shipmentBolSampleData, shipmentBolTemplate } from '@templara/templates';

const document = renderDocument({
  template: shipmentBolTemplate,
  data: shipmentBolSampleData,
  mode: 'preview',
});

export function BolPreview() {
  return <DocumentPreview document={document} scale={0.6} />;
}
```
